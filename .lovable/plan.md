
# План: Премиум B2B лендинг "Оплата счетов в Китай"

Соберу одностраничный лендинг в стиле Stripe/Mercury/Deel на русском языке. Все секции — отдельные компоненты, маршрут — главная страница (`src/routes/index.tsx`).

## Дизайн-система

Обновлю `src/styles.css` — задам семантические токены под палитру:
- Dark blue (основной фон/акценты): глубокий navy `oklch(~0.22 0.05 260)`
- Graphite (текст, поверхности): `oklch(~0.30 0.02 260)`
- White / off-white фон
- Accent green (CTA, индикаторы): свежий emerald `oklch(~0.72 0.17 155)`
- Градиенты: navy → graphite, тонкая зелёная подсветка
- Тени: мягкие, премиальные (elevation-1/2/3)
- Радиусы: 12–16px
- Шрифты: подключу через `<link>` в `__root.tsx` — **Manrope** (заголовки) + **Inter** (body). Крупные заголовки (clamp до ~72px), плотный tracking.

Все цвета — только через токены (никаких `text-white`, `bg-[#...]`).

## Структура секций (новые компоненты в `src/components/landing/`)

1. **Navbar** — лого "ПэйЧайна" (заглушка), пункты меню, кнопка "Оставить заявку". Sticky, blur-фон при скролле.
2. **Hero** — крупный заголовок, подзаголовок, две CTA, 3 trust-индикатора с зелёными чекмарками. Справа — премиальная иллюстрация (SVG-композиция: инвойс + поток платежа + китайский мотив, без стоковых клише). Тонкий градиент-фон, плавающая сетка.
3. **Advantages** — заголовок "Почему выбирают нас", 6 карточек с иконками (lucide-react: Building2, Globe2, FileCheck2, ShieldCheck, UserRoundCheck, LifeBuoy). Сетка 3×2, hover-эффект.
4. **HowItWorks** — 5 шагов "Как проходит оплата". Горизонтальная timeline на desktop, вертикальная на mobile. Номера в кружках, соединительная линия с градиентом.
5. **Services** — "Наши услуги", 5 элементов в формате list/cards с иконками.
6. **TrustBlock** — заголовок, описание, ниже статистика 3 цифры (1000+, 100+, "Персональное сопровождение"). Крупные цифры в акцентном цвете.
7. **FAQ** — accordion (shadcn) с 4 вопросами + готовые развёрнутые ответы на русском.
8. **LeadForm** — форма с 5 полями (react-hook-form + zod валидация), toast при отправке (sonner). Сабмит сейчас локальный (без бэкенда), показывает success-state. Слева — копирайт-блок "Обсудим вашу задачу" + контакты.
9. **Footer** — меню "Услуги / О компании / Контакты", контактный блок (Телефон, Email, Telegram с иконками), копирайт.

## Анимации

- Framer-motion (если не установлен — добавлю `bun add motion`): fade-up на скролл для секций, stagger для карточек, hover-lift.
- Tailwind animate утилиты для мелочей.

## Иллюстрации/иконки

- Иконки: `lucide-react` (уже есть).
- Hero-иллюстрация: сгенерирую через imagegen (премиум, прозрачный фон) — абстрактная композиция: документ-инвойс, стрелка платежа, элементы карты Китая/мост, в палитре navy + green. Без людей и фотостока.
- Возможно второе изображение для Trust-блока (или чисто типографика + градиент).

## SEO / `__root.tsx`

- Title: "Оплата счетов в Китай для бизнеса | Сопровождение ВЭД"
- Meta description (<160 симв) на русском.
- og:title, og:description, twitter:card.
- lang="ru" в `<html>`.
- Подключение Google Fonts через `<link>` в head root-роута.

## Технические детали

- Маршрут: `src/routes/index.tsx` — собирает все секции.
- Компоненты: `src/components/landing/{Navbar,Hero,Advantages,HowItWorks,Services,TrustBlock,FAQ,LeadForm,Footer}.tsx`.
- Использую существующие shadcn: `button`, `card`, `accordion`, `input`, `textarea`, `label`, `form`, `sonner`.
- Адаптив: mobile-first, breakpoints sm/md/lg, гамбургер-меню в Navbar.
- Без бэкенда — форма не сохраняет данные (можно подключить Lovable Cloud позже, спрошу при необходимости).

## Что НЕ делаю на этом шаге

- Бэкенд/БД/реальная отправка формы — только UI + toast (могу добавить Lovable Cloud отдельным шагом).
- Мультиязычность — только русский.
- Отдельные роуты для "Услуги/О компании/Контакты" — пока это якоря на лендинге (если попросите полноценные страницы, добавлю отдельно).
